# Real-Time Analytics Dashboard (Product Metrics)

A product-metrics platform that ingests events in real time, aggregates KPIs, and serves dashboards for business and engineering.

## Planned Features
- Event ingestion (API + Kafka)
- Stream processing / aggregation
- Metrics API (KPIs, trends)
- Dashboard UI (charts + filters)
- Auth + RBAC (optional)

## Tech Stack (planned)
- Backend: Java 17, Spring Boot
- Messaging: Kafka
- Storage: PostgreSQL (metrics), Redis (cache)
- UI: React or Angular
- Infra: Docker Compose
